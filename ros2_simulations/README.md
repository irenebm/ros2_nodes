# ROS 2 Simulations


## Launch

### ROSbot XL in Gazebo

```shell
ros2 launch ros2_simulations gazebo_rosbot_xl.launch.py
```

Now you can move around the environment with ``Nav2`` and create a map!
Click the “Nav2 Goal” button and choose a destination.
